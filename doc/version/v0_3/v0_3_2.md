# v0.3.2：Reference 類型

<br>

---

<br>

**目標**：實現 `java.lang.ref` 套件中的引用類型

**設計要點**：

```
四種引用強度（從強到弱）：

Strong Reference（強引用）
    │  Object obj = new Object();
    │  只要強引用存在，物件不會被回收
    ▼
Soft Reference（軟引用）
    │  SoftReference<Object> ref = new SoftReference<>(obj);
    │  記憶體不足時才會被回收
    ▼
Weak Reference（弱引用）
    │  WeakReference<Object> ref = new WeakReference<>(obj);
    │  下次 GC 時就會被回收
    ▼
Phantom Reference（虛引用）
       PhantomReference<Object> ref = new PhantomReference<>(obj, queue);
       無法通過虛引用取得物件，僅用於追蹤物件被回收的時機
```

<br>

**需要實現的類別**：

| 類別 | 功能 |
|------|------|
| `Reference<T>` | 引用基類 |
| `SoftReference<T>` | 軟引用 |
| `WeakReference<T>` | 弱引用 |
| `PhantomReference<T>` | 虛引用 |
| `ReferenceQueue<T>` | 引用佇列 |
