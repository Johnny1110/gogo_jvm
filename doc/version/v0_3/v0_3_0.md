# v0.3.0：反射基礎

<br>

---

<br>

**目標**：實現 `java.lang.Class` 物件與基本反射功能

**設計要點**：

```
每個類別在 JVM 中都有對應的 Class 物件：

  ┌─────────────────┐         ┌─────────────────┐
  │ String 實例      │         │ Integer 實例    │
  │ ┌─────────────┐ │         │ ┌─────────────┐ │
  │ │class pointer│─┼────┐    │ │class pointer│─┼────┐
  │ └─────────────┘ │    │    │ └─────────────┘ │    │
  └─────────────────┘    │    └─────────────────┘    │
                         ▼                           ▼
                 ┌──────────────┐            ┌──────────────┐
                 │ Class<String>│            │Class<Integer>│
                 │ (Class 物件) │            │ (Class 物件) │
                 └──────────────┘            └──────────────┘
```

<br>

**需要實現的功能**：

| 功能 | 說明 |
|------|------|
| `Object.getClass()` | 取得物件的 Class |
| `Class.forName()` | 根據名稱載入類別 |
| `Class.getName()` | 取得類別名稱 |
| `Class.getSuperclass()` | 取得父類 |
| `Class.getInterfaces()` | 取得實現的介面 |
| `Class.newInstance()` | 反射創建實例 |

<br>

**測試目標**：

```java
public class TestReflection {
    public static void main(String[] args) {
        Object obj = new String();
        Class<?> clazz = obj.getClass();
        System.out.println(clazz.getName());  // java.lang.String
    }
}
```
