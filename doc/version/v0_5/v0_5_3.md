# v0.5.3：Copying GC

<br>

---

<br>


**目標**：實現複製式垃圾回收器，適用於 Young Generation

**Semi-Space 設計**：

```
Copying GC（Cheney 演算法）：

GC 前：
  From Space              To Space
  ┌───┬───┬───┬───┬───┐  ┌───────────────┐
  │ A │ B │ C │ D │ E │  │   (空的)       │
  └───┴───┴───┴───┴───┘  └───────────────┘
    ↑       ↑
  存活    垃圾

GC 後：
  From Space (現在是 To)   To Space (現在是 From)
  ┌───────────────┐       ┌───┬───┬───────┐
  │   (空的)       │       │ A │ C │ Free  │
  └───────────────┘       └───┴───┴───────┘
                            只複製存活物件

優點：
- 沒有碎片
- 分配速度快（Bump Pointer）
- 適合存活率低的場景（Young Generation）

缺點：
- 記憶體利用率只有 50%
- 存活物件多時效率低
```
